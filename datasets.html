CloudDrift provides convenience functions to access real-world ragged-array datasets. Currently available datasets are:

Global Drifter Program (GDP) Datasets#

  • clouddrift.datasets.gdp1h(): 1-hourly Global Drifter Program (GDP) data from a cloud-optimized Zarr dataset on AWS.

  • clouddrift.datasets.gdp6h(): 6-hourly GDP data from zarr archive hosted on a public AWS S3 bucket accessible at s3://noaa-oar-hourly-gdp-pds/experimental/.

  • clouddrift.datasets.gdp_source(): source GDP data without being pre-processed unlike the 6-hourly/1-hourly datasets that are derived from it.

You can find examples of how to use the GDP datasets in the GDP Get-Started Notebooks.

GLAD Dataset#

clouddrift.datasets.glad(): 15-minute Grand LAgrangian Deployment (GLAD) data produced by the Consortium for Advanced Research on Transport of Hydrocarbon in the Environment (CARTHE) and hosted upstream at the Gulf of Mexico Research Initiative Information and Data Cooperative (GRIIDC).

You can find examples of how to use the GLAD datasets in the GLAD Get-Started Notebooks.

MOSAiC Dataset#

clouddrift.datasets.mosaic(): MOSAiC sea-ice drift dataset as a ragged array processed from the upstream dataset hosted at the NSF’s Arctic Data Center.

You can find examples of how to use the MOSAiC datasets in the MOSAiC Get-Started Notebook.

HURDAT2 Dataset#

clouddrift.datasets.hurdat2(): The HURricane DATa 2nd generation (HURDAT2) processed from the upstream dataset hosted at the NOAA AOML Hurricane Research Division.

You can find examples of how to use the HURDAT2 datasets in the HURDAT2 Get-Started Notebook.

IBTrACS Dataset#

clouddrift.datasets.ibtracs(): The International Best Track Archive for Climate Stewardship (IBTrACS) as a ragged array xarray dataset. Processed from the upstream dataset hosted at the NOAA NCEI IBTrACS.

You can find examples of how to use the IBTrACS datasets in the IBTrACS Get-Started Notebook.

Andro Dataset#

clouddrift.datasets.andro(): The ANDRO dataset as a ragged array processed from the upstream dataset hosted at the SEANOE repository.

Subsurface Floats Dataset#

clouddrift.datasets.subsurface_floats(): The subsurface float trajectories dataset as hosted by NOAA AOML at NOAA’s Atlantic Oceanographic and Meteorological Laboratory (AOML) and maintained by Andree Ramsey and Heather Furey from the Woods Hole Oceanographic Institution.

Spotters Dataset#

clouddrift.datasets.spotters(): The Sofar Ocean Spotters archive dataset as hosted at the public AWS S3 bucket.

YoMaHa’07 Dataset#

clouddrift.datasets.yomaha(): The YoMaHa’07 dataset as a ragged array processed from the upstream dataset hosted at the Asia-Pacific Data-Research Center (APDRC).

More Information#

The GDP and the Spotters datasets are accessed lazily, so the data is only downloaded when specific array values are referenced. The ANDRO, GLAD, MOSAiC, Subsurface Floats, and YoMaHa’07 datasets are downloaded in their entirety when the function is called for the first time and stored locally for later use.
